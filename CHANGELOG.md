# Changelog

Заметные изменения пользовательской части dhqclash.

## 2026-07-08
- Публичная витрина переехала в новый репозиторий `dhqclash-public`; ссылки
  на release `custom-mrs` и Issue-форму обновлены. Клиентские конфиги
  подтягивают новый адрес правил автоматически.

- Исходный код и инфраструктура вынесены в приватный репозиторий; публичный
  репозиторий теперь содержит пользовательскую документацию, Issue-форму
  для правил маршрутизации и release `custom-mrs` со скомпилированными правилами.
- Голые имена в правилах (например, `ozon`) разворачиваются в маску по всем
  поддоменам и TLD.
- Бот: срок подписки в `/config`, переименование устройств `/rename`.
